Context:
Settings icon in Monitor 1 and Monitor 2 panels

----------------------------------------------

Primary UI Element (Settings Icon):
- Remove the preset dropdown and label in the panel header area (this has to be moved to a new pop-up)
- Add a small settings icon (gear icon) to the top-right corner of the panels (both in Monitor 1 and Monitor 2).
- The icon must appear inside the panel header area.
⁃ Icon size: small (toolbar-style, approx. 16x16 or 20x20 pixels)

----------------------------------------------

Interaction (Settings Icon):
- Enable click event on the icon
- When the user clicks the settings icon, open a popup dialog
- The popup should be anchored to the panel (not full-screen)

----------------------------------------------

Secondary UI Element (Popup / Dialog):

   Label: "Max No. of Verses / Slide"

3. Textbox 1:
   Label: "Max No. of Characters / Slide

4. Dropdown 3:
   Label: "Format"
   Options: ['Verse', 'Paragraph']
